# Admission Lead Management System

A responsive web-based application developed to manage and track student admission leads efficiently. The system helps organize student inquiries, maintain lead records, and improve the admission follow-up process through an interactive and user-friendly interface.

---

## Features

- Add and manage admission leads
- Store student details and contact information
- Responsive and user-friendly interface
- Dynamic form validation using JavaScript
- Organized lead tracking system
- Simple and clean UI design
- Easy navigation and data handling

---

## Technologies Used

- **HTML5** – Structure of the web application
- **CSS3** – Styling and responsive design
- **JavaScript** – Dynamic functionality and validation

## How to Run the Project

### 1. Clone the Repository

```bash
git clone https://github.com/Kabilroy/Admission-Lead-Management-System.git
```

### 2. Open the Project Folder

```bash
cd Admission-Lead-Management-System
```

### 3. Run the Project

Open `index.html` in your browser.
